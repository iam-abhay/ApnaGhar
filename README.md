## 📖 Project Overview

ApnaGhar is a professional real estate landing page designed to demonstrate modern frontend development skills. It focuses on clean UI, responsive layouts, SEO optimization, accessibility, and lightweight performance without relying on external frameworks.

This project is suitable for showcasing frontend development skills to recruiters and hiring managers.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/iam-abhay/ApnaGhar.git
cd ApnaGhar
```

### Run Locally

You can simply open `index.html` in your browser, or start a local server.

Using Python:

```bash
python -m http.server 8000
```

Open:

```
http://localhost:8000
```

---

## 🌍 Deployment

This project is optimized for **GitHub Pages**.

1. Push your project to GitHub.
2. Open your repository.
3. Go to **Settings → Pages**.
4. Under **Build and Deployment**:
   - Source → **Deploy from a branch**
   - Branch → **main**
   - Folder → **/(root)**
5. Click **Save**.

Your website will be available at:

```
https://iam-abhay.github.io/ApnaGhar/
```
